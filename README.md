# PRODIGY_WD_01

Responsive Landing Page

## Description
This project is a responsive landing page created as part of Task-01 of the internship.
It includes a fixed navigation bar that changes its style when the page is scrolled
and when a user hovers over menu items.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Fixed navigation menu visible on all sections
- Hover effect on navigation items
- Navigation bar color change on scroll
- Responsive design for different screen sizes
